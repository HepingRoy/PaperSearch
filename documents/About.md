# 项目规划
## About Project
PaperSearch是一个B/S架构的Paper内容检索服务。提供Paper上传，内容检索，定制化搜索，paper隔离，paper搜索推荐等功能。
## Core Artifacts
- PaperSearch 技术框架
  - Solr
  - SpringMVC
  - AntDesign+ReactJS
- 关键技术
  - 分词
  - 搜索推荐
  - 搜索排名
